Jours 1: Comprendre les bases de HTML

Introduction à HTML:

Comprendre ce qu'est HTML et son rôle dans le développement web.
Apprendre la structure de base d'un document HTML.
Balises et Éléments:

Connaître les balises HTML courantes (par exemple, <html>, <head>, <body>).
Comprendre la différence entre les balises de conteneur et les balises autonomes.
Création d'une Page HTML Simple:

Écrire le squelette de base d'un document HTML.
Ajouter des éléments tels que le titre, les paragraphes, et les en-têtes.



Jour 2:


Liens et Attributs:

Utiliser les balises <a> pour créer des liens hypertexte.
Explorer les attributs HTML courants (par exemple, href, src).
Listes et Tables:

Créer des listes ordonnées et non ordonnées.
Construire des tableaux basiques avec les balises <table>, <tr>, <td>.
Formulaires HTML:

Introduire les formulaires avec les balises <form>, <input>, <button>.
Comprendre les différents types d'entrées de formulaire (texte, mot de passe, boutons radio, etc.).
Activité pratique:
Projet 1: Créer une page HTML simple présentant quelques informations personnelles. Ajouter des liens vers tes réseaux sociaux et utiliser une liste pour afficher quelques compétences.
